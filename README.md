## Deskripsi Proyek
Repository ini berisi proyek **Ujian Akhir Semester (UAS)** mata kuliah **Pemrograman Basis Data**. Proyek yang dikembangkan adalah **Sistem Basis Data Penjualan** yang bertujuan untuk menerapkan konsep perancangan dan implementasi basis data menggunakan MySQL sesuai dengan materi yang terdapat pada **Jurnal Kuliah Pemrograman Basis Data**.

Sistem ini mencakup perancangan ERD, normalisasi data, implementasi struktur tabel, pengisian data, serta penggunaan query SQL lanjutan seperti JOIN, GROUP BY, HAVING, dan agregasi.

## Tujuan Proyek
Tujuan dari pembuatan proyek ini adalah:
- Menerapkan konsep perancangan basis data menggunakan ERD dan relasi antar tabel
- Melakukan normalisasi data hingga bentuk normal ketiga (3NF)
- Mengimplementasikan basis data menggunakan MySQL
- Menggunakan perintah SQL meliputi DDL, DML, dan TCL
- Mengimplementasikan query lanjutan sesuai ketentuan UAS

## Cara Menjalankan Database
1. Pastikan MySQL telah terinstal pada komputer
2. Buka MySQL Workbench
3. Buat atau pilih koneksi database
4. Jalankan file `uas_pbd_sistem_penjualan.sql` yang terdapat pada folder **SQL**
5. Pastikan database dan seluruh tabel berhasil dibuat tanpa error

## Fitur SQL yang Digunakan
- **DDL (Data Definition Language)**  
  CREATE DATABASE, CREATE TABLE, PRIMARY KEY, FOREIGN KEY
- **DML (Data Manipulation Language)**  
  INSERT, SELECT
- **TCL (Transaction Control Language)**  
  START TRANSACTION, COMMIT
- **Query Lanjutan**  
  JOIN, GROUP BY, HAVING, fungsi agregasi (SUM)
